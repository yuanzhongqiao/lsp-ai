<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">LSP-AI 是一款开源</font></font><a href="https://microsoft.github.io/language-server-protocol/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">语言服务器</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，可作为后端，使用大型语言模型执行补全，并很快推出其他支持 AI 的功能。由于它是一款语言服务器，因此可与任何支持 LSP 的编辑器配合使用。</font></font></p>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">LSP-AI 的目标是通过与软件工程师已经熟悉和喜爱的工具集成来协助和增强软件工程师的能力，而不是取代软件工程师。</font></font></strong></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">与其合作的部分编辑人员名单如下：</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">VS 代码</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">NeoVim</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Emacs</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">螺旋</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">升华</font></font></li>
</ul>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">它可以与很多很多编辑器兼容。</font></font></p>

<div class="markdown-heading" dir="auto"><h1 tabindex="-1" class="heading-element" dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">LSP-AI 案例</font></font></h1><a id="user-content-the-case-for-lsp-ai" class="anchor" aria-label="永久链接：LSP-AI 案例" href="#the-case-for-lsp-ai"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a></div>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">tl;dr LSP-AI 从编辑器特定的插件作者那里抽象出复杂的实现细节，将开源开发工作集中到一个可共享的后端。</font></font></strong></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">编辑器集成的人工智能助手将会继续存在。它们并不完美，但正在不断改进，</font></font><a href="https://arxiv.org/pdf/2206.15331" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">早期研究已经显示出其优势。虽然有几家公司已经发布了像</font></font></a><font style="vertical-align: inherit;"></font><a href="https://cursor.sh/" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Cursor</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">这样的先进人工智能编辑器</font><font style="vertical-align: inherit;">，但开源社区缺乏直接竞争对手。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">LSP-AI 旨在通过提供语言服务器来填补这一空白，该服务器将 AI 驱动的功能集成到我们熟知和喜爱的编辑器中。以下是我们认为 LSP-AI 是必要且有益的原因：</font></font></p>
<ol dir="auto">
<li>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">统一AI功能</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">：</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">通过将 AI 功能集中到单个后端，LSP-AI 允许受支持的编辑器从这些进步中受益，而无需多余的开发工作。</font></font></li>
</ul>
</li>
<li>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">简化插件开发</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">：</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">LSP-AI 简化了设置 LLM 后端、构建复杂提示等复杂操作。插件开发人员可以专注于增强他们正在开发的特定编辑器，而不必处理后端的复杂性。</font></font></li>
</ul>
</li>
<li>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">增强协作</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">：</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">提供共享后端可创建一个协作平台，让开源开发人员可以齐聚一堂，共同添加新功能。这种统一的努力可以促进创新并减少重复工作。</font></font></li>
</ul>
</li>
<li>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">广泛兼容性</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">：</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">LSP-AI 支持任何遵守语言服务器协议 (LSP) 的编辑器，确保广泛的编辑器可以利用 LSP-AI 提供的 AI 功能。</font></font></li>
</ul>
</li>
<li>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">灵活的 LLM 后端支持</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">：</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">目前，LSP-AI 支持 llama.cpp、Ollama、与 OpenAI 兼容的 API、与 Anthropic 兼容的 API、与 Gemini 兼容的 API 以及与 Mistral AI FIM 兼容的 API，让开发人员可以灵活地选择他们喜欢的后端。这个列表很快就会增加。</font></font></li>
</ul>
</li>
<li>
<p dir="auto"><strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">面向未来</font></font></strong><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">：</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">LSP-AI 致力于紧跟 LLM 驱动软件开发的最新进展。</font></font></li>
</ul>
</li>
</ol>
